## 关于

`xInitialize` 致力于做一个即开即用的中后端解决方案，可作为新项目的启动模板！

本项目由以下两部分组成：

- [xInitialization/xAdmin](https://github.com/xInitialization/xAdmin) - 前端应用
- [xInitialization/xBackstage](https://github.com/xInitialization/xBackstage) - 后端服务
